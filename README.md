A Python program that performs various operations on **NumPy arrays**.

- Calculate **mean, variance, and standard deviation**

- Perform different **array manipulation operations**

### 🔹 2. Array Operations

Programs demonstrating different **array manipulations using NumPy**.

**Examples**
